# FizzBuzz

A classic programming challenge that prints numbers from 1 to 100, replacing multiples of 3 with "Fizz", multiples of 5 with "Buzz", and multiples of both 3 and 5 with "FizzBuzz".

## Overview

This is a simple Python implementation of the FizzBuzz problem. It prints the numbers from 1 to 100, but:
- For multiples of 3, it prints "Fizz".
- For multiples of 5, it prints "Buzz".
- For multiples of both 3 and 5, it prints "FizzBuzz".

## How to Use

1. Run the script to see the output of the FizzBuzz sequence.

## Features

- Classic FizzBuzz logic.
- Prints the numbers and special cases in a simple, readable format.
